# 内容安全

DemoGo 会在公开链接生成前执行内容检查。

## 基本原则

内容安全不应阻断正常市场营销行为。

允许：

- 姓名、手机号、邮箱、公司名称。
- 报名、预约、咨询、试用申请、留言。
- 正常引流和获客页面。

拦截：

- 诈骗、违法交易、博彩、色情低俗。
- 恶意下载、高风险金融诱导。
- 收集身份证号、银行卡号、验证码、密码、人脸照片等高敏信息。
- 明显规避监管或违法用途。

## 当前能力边界

当前主要检查：

- 页面源码。
- 文本内容。
- 构建后的发布目录。
- 部分可疑文件名。

当前不能可靠识别：

- 图片、视频、音频里的真实内容。
- 复杂动态运行后的页面状态。

## 失败处理

如果内容检查失败：

1. 告诉用户为什么不能发布。
2. 给出“让 AI 怎么改”的明确建议。
3. 正常营销表单不要误删，只删除真正高风险字段或表达。
4. 不允许绕过内容检查。

## 后续增强

- 管理端人工复核闭环继续完善。
- 真实试用后根据误判情况调整规则。
- 当本地规则不足时，再接入第三方内容安全服务。
