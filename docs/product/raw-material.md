# DemoGo OPC 比赛 PPT 原料

## 项目定位
DemoGo (v0.5.0) — AI编程产物一键发布与试用平台
域名: https://demogo.cn
一句话: "AI 生成代码 → 一键发布 → 马上试用"

## 核心能力
- 静态页面托管、前端源码构建、Node.js Docker运行时、MySQL试用数据库自动分配
- 17种项目类型自动识别 (React/Vue/Vite/Next.js/Nuxt/Astro/Express/NestJS 等)
- CLI/MCP/Skill/API 四层AI工具集成
- 12类内容安全审查，fail-closed策略
- Free/Lite/Pro 三级套餐

## 受众与场景
- OPC大赛评委，需要了解项目价值、技术创新、商业前景
- 项目路演场景，需要简洁有力、数据支撑
