## 素材制作建议

| 图表类型 | 建议工具 | 输出格式 |
|---------|---------|---------|
| 趋势图 / 柱状图 | Figma / ECharts / Chart.js → 导出 PNG | PNG @2x |
| 漏斗图 | Figma / Excalidraw | SVG |
| 架构图 | Excalidraw / Draw.io / Mermaid | SVG |
| 竞品对比表 | 直接 HTML PPT 内建表格 | 内建 |
| 饼图 | Figma / ECharts | PNG @2x |
| 时间线 | Figma / 纯 CSS | SVG / 内建 |
| 截图 | 浏览器自带截图 / Greenshot | PNG @2x 无损 |
