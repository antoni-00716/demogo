# DemoGo Web 前端

React + TypeScript + Vite 构建的 DemoGo 前端应用。

## 目录结构

```
web/
├── src/
│   ├── api/          API 请求层
│   ├── assets/       静态资源
│   ├── components/   组件
│   ├── config/       配置
│   ├── hooks/        Hooks
│   ├── pages/        页面（admin、dashboard）
│   ├── stores/       状态管理
│   ├── styles/       样式
│   └── utils/        工具函数
├── vite.config.ts
└── package.json
```

## 本地开发

```bash
cd web
npm install
npm run dev
```

## 检查

```bash
npm run lint
npm run build
```
