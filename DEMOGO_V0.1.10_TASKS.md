# DemoGo v0.1.10 开发任务清单

更新时间：2026-05-12

## 1. 版本目标

v0.1.10 是“试用转化闭环修复版”。

核心目标：

- 用户能主动申请升级 Lite / Pro；
- 管理员能在后台查看、筛选、开通、拒绝升级申请；
- 转发文案不再使用浏览器弹窗；
- 继续参考国际成熟平台，完成一小步架构迁移；
- 保持 v0.1.9 已上线能力不受影响。

## 2. P0 功能任务

### A. 用户申请升级套餐

- [x] 用户端新增“申请升级套餐”入口；
- [x] 支持选择 `lite` / `pro`；
- [x] 支持填写联系方式；

- [x] 写入审计日志 `request_plan_upgrade`。

### B. 管理后台处理升级申请

- [x] 管理后台新增“升级申请”模块；
- [x] 支持关键词搜索；

- [x] 处理动作写入审计日志；
- [x] 后台运营摘要显示待处理升级申请数。

### C. 转发文案体验修复

- [x] 用户端复制转发文案不再使用浏览器弹窗；

- [x] 一键复制失败时显示可手动复制的文本框；
- [x] 支持选中文案；
- [x] 支持收起文案面板；
- [x] 文案不夸大 DemoGo 能力。
