## v2 → v3 改动总结

| 问题 | 怎么改的 |
|------|---------|
| "不用X、不用Y、不用Z"排比生硬 | 拆成短句，或者换说法（"不用下载，不用切窗口，不用再开一个网页"→ 虽然还是排比但每个都短且具体） |
| 抽象词（实现、产出、访问、部署） | 全换掉。实现 → 做出来 / 搞定；产出 → 做；访问 → 点开看；部署 → 不提 |
| 句子太长读不顺 | 全部拆到 15 字以内。一句话只说一件事 |
| 主语缺失（"或者拖进来"） | 补上："下到本地了？拖进去也行" |
| 语气像"写文案"不像"说话" | 每句读出声过一遍，拗口的全改 |
| "一切就这么简单"这类空话 | 删掉。用具体描述替代 |
| 对比表"自己部署" → "自己搞" | "部署"这个词就不该出现 |
| Footer tagline"AI 产出的作品" | 改成"用 AI 做的东西" |
